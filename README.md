# Wiznu For Hair

A refresh\restyle of the website for 'Wiznu for Hair' (wiznu.com.au)

Built with HTML5, CSS3, JS & PHP

Utilizing Bootstrap 5.3.2, Font Awesome 6.4, Google ReCaptcha v2, SwiperJS, Headroom.js, NanoGallery2 & Wow.JS
